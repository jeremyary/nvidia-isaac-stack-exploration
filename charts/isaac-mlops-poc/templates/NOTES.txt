Isaac MLOps PoC deployed to namespace: {{ include "isaac-mlops.namespace" . }}

=== Post-install steps ===

1. Link NGC pull secret to the DSPA pipeline runner SA:

   oc secrets link pipeline-runner-isaac-pipelines ngc-secret \
     --for=pull -n {{ include "isaac-mlops.namespace" . }}

2. Verify infrastructure is ready:

   oc get pods -n {{ include "isaac-mlops.namespace" . }}
   # Expected: minio, mariadb-*, ds-pipeline-* pods Running

3. Compile and upload the pipeline:

   python pipelines/isaac_training_pipeline.py
   # Upload pipelines/isaac_training_pipeline.yaml via the RHOAI dashboard
