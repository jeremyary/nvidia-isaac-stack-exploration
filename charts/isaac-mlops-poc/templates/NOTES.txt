Isaac MLOps PoC deployed to namespace: {{ include "isaac-mlops.namespace" . }}

=== Post-install steps ===
{{- if not .Values.ngc.apiKey }}

1. Create NGC secrets and link to the pipeline runner SA:

   source .env
   oc create secret docker-registry ngc-secret \
     --docker-server=nvcr.io --docker-username='$oauthtoken' \
     --docker-password="$NGC_API_KEY" --docker-email="{{ .Values.ngc.email }}" \
     -n {{ include "isaac-mlops.namespace" . }}
   oc create secret generic ngc-api-key \
     --from-literal=NGC_API_KEY="$NGC_API_KEY" \
     -n {{ include "isaac-mlops.namespace" . }}
   oc secrets link pipeline-runner-isaac-pipelines ngc-secret \
     --for=pull -n {{ include "isaac-mlops.namespace" . }}

   Tip: pass --set ngc.apiKey=<key> at install time to automate this.
{{- else }}

1. NGC secrets created automatically (ngc.apiKey was provided).
   Link the pull secret to the pipeline runner SA:

   oc secrets link pipeline-runner-isaac-pipelines ngc-secret \
     --for=pull -n {{ include "isaac-mlops.namespace" . }}
{{- end }}

2. Verify infrastructure is ready:

   oc get pods -n {{ include "isaac-mlops.namespace" . }}
   # Expected: minio, mariadb-*, ds-pipeline-* pods Running

3. Compile and upload the pipeline:

   python pipelines/isaac_training_pipeline.py
   # Upload pipelines/isaac_training_pipeline.yaml via the RHOAI dashboard
